Algorithm - Selection sort algorithm

Concept

假設有 n 個元素, 我們要幫它們排序, 該怎麼做？ 選擇排序演算法提供了一種方法. 先從 n 個元素找出最小的元素, 然後放在第1個位址. 接著, 再從剩下 n-1 元素中找出最小的元素, 放在第2個位址, 以此類推, 一直到最後一個元素為止.
